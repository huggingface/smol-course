# 提交你的最终项目！

是时候提交你的 DPO 对齐模型了！本单元使用与第 1 单元相同的基于排行榜的提交系统。以下是计划：

1. 阅读本章的书面指南 ✅
2. 使用你在本章学到的知识训练一个模型。
3. 将模型推送到 Hugging Face Hub。
4. 使用 `hf jobs` 评估模型。
5. 在排行榜上开启一个拉取请求。

在本页面中，我们将逐步完成每个步骤。

## 1. 阅读本章的书面指南 和 2. 使用你在本章学到的知识训练一个模型

对于第 2 单元的提交，你应该阅读本单元中的所有材料，并使用 DPO 训练一个偏好对齐模型。训练代码在以下位置提供：

- [DPO 训练练习](./4) - 使用 SmolLM3 的完整 DPO 训练指南

你需要将此与第 1 单元的[使用 Hugging Face Jobs 训练](../unit1/5)技术结合使用。

## 3. 将模型推送到 Hugging Face Hub

一旦你训练了 DPO 对齐模型，你需要将其推送到 Hugging Face Hub 上的一个仓库。如果你在训练命令中添加 `--push_to_hub` 标志，TRL 会为你处理这个问题。

### 带有 Hub 上传的 DPO 训练：
```bash
hf jobs uv run \
    --flavor a100-large \
    --secrets HF_TOKEN \
    "https://raw.githubusercontent.com/huggingface/trl/main/trl/scripts/dpo.py" \
    --model_name_or_path HuggingFaceTB/SmolLM3-3B \
    --dataset_name Anthropic/hh-rlhf \
    --learning_rate 5e-7 \
    --beta 0.1 \
    --max_steps 1000 \
    --push_to_hub \
    --hub_model_id your-username/smollm3-dpo-aligned \
    --report_to trackio
```

你训练的模型将在 `your-username/your-model-name` 可用。有关详细文档，请查看 `transformers` 的[检查点文档](https://huggingface.co/docs/transformers/trainer#checkpoints)。

## 4. 使用 `hf jobs` 评估模型

现在，我们将评估你的 DPO 对齐模型。我们将使用 `hf jobs` 来评估模型，并将其与 `lighteval` 结合使用。我们将把评估结果推送到 Hub 上的数据集。

> [!TIP]
> 对于 DPO 评估，我们使用测试对齐的有帮助性和安全性方面的任务，包括 `truthfulqa`、`gsm8k` 和其他以对齐为重点的基准。

```bash
hf jobs uv run \
    --flavor a10g-large \
    --with "lighteval[vllm]" \
    --secrets HF_TOKEN \
    lighteval vllm "model_name=<your-username>/<your-model-name>" \
    "lighteval|truthfulqa:mc2|0|0,lighteval|hellaswag|0|0,lighteval|arc:challenge|0|0" \
    --push-to-hub --results-org <your-username>
```

此命令将使用 `lighteval` 和 `vllm` 评估模型，并将结果保存到你定义的 Hugging Face Hub 上的数据集仓库。

> [!TIP]
> 我们在第 2 单元中专注于对齐评估，但在第 2 单元中我们更详细地探索评估。DPO 评估的关键基准包括：
> - TruthfulQA：测试真实和诚实的响应
> - GSM8K：数学推理和有帮助性
> - MMLU：跨领域的广泛知识和有帮助性

## 5. 在排行榜空间上开启一个拉取请求

你现在准备好将你的 DPO 对齐模型提交到排行榜了！你需要做两件事：

1. 将你的模型结果添加到 `submissions.json`
2. 在 PR 文本中分享你的训练和评估命令。

### 将你的模型结果添加到 `submissions.json`

在[排行榜空间](https://huggingface.co/spaces/smol-course/leaderboard/edit/main/submissions.json)上开启一个拉取请求以提交你的模型。你只需要添加模型信息和对你在上一步中创建的数据集的引用。我们将提取结果并将其显示在排行榜上。

```json
{
    "submissions": [

        ... // 现有提交

        {
            "username": "<your-username>",
            "model_name": "<your-model-name>",
            "chapter": "3",
            "method": "DPO",
            "submission_date": "<your-submission-date>",
            "results-dataset": "<your-results-dataset>",
            "base_model": "HuggingFaceTB/SmolLM3-3B",
            "preference_dataset": "Anthropic/hh-rlhf"
        }
    ]
}
```

### 在 PR 文本中分享你的训练和评估命令

在 PR 文本中，分享你的训练和评估命令。

### 等待 PR 合并

一旦 PR 合并，你的 DPO 对齐模型将被添加到排行榜！你可以在[这里](https://huggingface.co/spaces/smol-course/leaderboard)查看排行榜。
