# 欢迎来到 🤗 smol-course

![Fine-Tuning Course thumbnail](https://github.com/huggingface/smol-course/blob/main/banner.png?raw=true)

欢迎来到**语言模型微调**的全面（且最小）课程！

这门免费课程将带你踏上旅程，**从初学者到专家**，理解、实现和优化大型语言模型的微调技术。

第一单元将帮助你入门：

* 发现**课程大纲**。
* **获取有关认证流程和时间表的更多信息**。
* 了解课程背后的团队。
* 创建你的**账户**。
* **注册我们的 Discord 服务器**，并认识你的同学和我们。

让我们开始吧！

> [!TIP]
> 这门课程虽小但节奏很快！它面向希望快速掌握 LLM 微调技能的软件开发人员和工程师。如果不是你，请查看 [LLM 课程](https://huggingface.co/learn/llm-course/)。

## 从这门课程中期待什么？

在本课程中，你将：

* 📖 学习指令微调、监督微调、偏好对齐、评估、视觉语言模型……等等！
* 🧑‍💻 学习使用成熟的微调框架和工具，如 TRL 和 Transformers。
* 💾 分享你的项目并探索社区创建的微调应用。
* 🏆 参加挑战，在挑战中你将评估你的微调模型与其他学生的模型。
* 🎓 通过完成作业获得结业证书。

在本课程结束时，你将了解**如何有效微调语言模型并使用最新的微调技术构建专业的 AI 应用**。

不要忘记[**注册课程！**](https://huggingface.co/smol-course)

## 课程是什么样的？

课程由以下部分组成：

* _基础单元_：你将学习微调**理论概念**。
* _实践_：你将学习**使用成熟的微调框架**来适应你的模型。这些实践部分将有预配置的环境。
* _用例作业_：你将应用所学概念来解决你选择的实际问题。
* _合作_：我们与 Hugging Face 的合作伙伴合作，为你提供最新的微调实现和工具。

这门**课程是一个不断发展的项目，随着你的反馈和贡献而演变！**欢迎在 GitHub 中开启 issue 和 PR，并在我们的 Discord 服务器中参与讨论。

## 先决条件是什么？

要能够学习本课程，你应该具备：

* AI 和 LLM 概念的基本理解
* 熟悉 Python 编程和机器学习基础
* 使用 PyTorch 或类似深度学习框架的经验
* 理解 transformer 架构基础

如果你没有这些，不用担心。查看 [LLM 课程](https://huggingface.co/learn/llm-course/)以开始学习。

> [!TIP]
> 上述课程本身不是先决条件，因此如果你理解 LLM 和 transformer 的概念，你现在就可以开始本课程！

## 我需要什么工具？

你只需要 2 样东西：

* _一台计算机_，有互联网连接，最好有 GPU 访问（[Hugging Face Pro](https://huggingface.co/pro) 效果很好）。
* _一个账户_：访问课程资源并创建项目。如果你还没有账户，可以在[这里](https://huggingface.co/join)创建一个（免费）。

## 认证流程

你可以选择以_旁听模式_学习本课程，或者完成活动并_获得我们将颁发的两个证书之一_。如果你旁听课程，你可以参加所有挑战并在需要时完成作业，**你不需要通知我们**。

认证流程**完全免费**：

* _要获得基础认证_：你需要完成课程的第 1 单元。这适用于希望了解指令微调基础而不构建高级应用的学生。
* _要获得结业证书_：你需要完成所有课程单元并提交最终项目。这适用于希望展示微调技术精通的学生。
