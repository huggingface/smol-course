欢迎来到最小的微调课程！本模块将指导你使用 **SmolLM3** 进行指令微调。SmolLM3 是 Hugging Face 最新的 30 亿参数模型，在同等规模下实现了最先进的性能，同时保持了易于学习和实验的特点。

## 什么是指令微调？

指令微调 (Instruction Tuning) 是使预训练语言模型适应遵循人类指令和参与对话的过程。虽然像 [`SmolLM3-3B-Base`](https://huggingface.co/HuggingFaceTB/SmolLM3-3B-Base) 这样的基础模型被训练来预测下一个词元 (Token)，但像 [`SmolLM3-3B`](https://huggingface.co/HuggingFaceTB/SmolLM3-3B) 这样经过指令微调的模型被专门训练来：

- 准确遵循用户指令
- 进行自然对话
- 提供有帮助、无害且诚实的响应
- 在多轮交互中保持上下文
- 使用工具或 MCP 服务器执行任务

从文本补全模型转变为遵循指令的助手，这一转变是通过在精心策划的数据集上进行监督微调 (Supervised Fine-Tuning) 来实现的。

> [!TIP]
> 我们在大语言模型课程中深入探讨了指令微调，请参见[这里](https://huggingface.co/learn/llm-course/en/chapter11/2)。

## 为什么选择 SmolLM3 来学习？

[SmolLM3](https://hf.co/blog/smollm3) 非常适合学习指令微调，因为它：

- 可以在单个 GPU 上以合理的成本运行
- 实现了有竞争力的性能
- 支持多语言对话
- 支持最长 8k 词元的扩展上下文长度（某些变体支持最长 128k 词元的上下文）
- 具有双模式推理能力，具备明确的思考能力
- 配有完整的训练配方，让你完全理解它是如何构建的

## 模块概览

在这个综合模块中，我们将探索四个关键领域：

### 对话模板

对话模板 (Chat Templates) 是指令微调的基础——它们构建用户和 AI 模型之间的交互结构，确保一致且符合上下文的响应。你将学习：

- SmolLM3 的对话模板如何工作
- 将对话转换为正确的格式
- 使用系统提示词和多轮对话
- 使用 Transformers 库内置的模板支持

详细信息请参见[对话模板](./2)。

### 监督微调 (SFT)

监督微调 (Supervised Fine-Tuning, SFT) 是使预训练模型适应遵循指令的核心技术。你将掌握：

- SFT 背后的理论以及何时使用它
- 使用 SmolTalk2 数据集
- 使用 TRL 的 [`SFTTrainer`](https://huggingface.co/docs/trl/en/sft_trainer) 进行高效训练
- 数据准备和训练配置的最佳实践

全面指南请参见[监督微调](./3)。

### 实践练习

通过逐步提升难度的练习将知识付诸实践：

- 为指令微调处理数据集
- 在不同任务上微调 SmolLM3
- 使用 Python API 和 CLI 工具
- 比较基础模型与微调模型的性能

完整的练习和示例请参见[练习](./4)。

### Hugging Face Jobs

Hugging Face [Jobs](https://huggingface.co/docs/huggingface_hub/en/guides/jobs) 是一个完全托管的云基础设施，用于训练模型，无需设置 GPU、管理依赖项或在本地配置环境的麻烦。这对于 SFT 训练特别有价值，因为它可能需要大量资源和时间。

全面指南请参见 [Hugging Face Jobs](./5)。

## 你将构建什么

在本模块结束时，你将：

- 在自定义数据集上微调你自己的 SmolLM3 模型
- 理解对话模板格式和对话结构
- 获得使用编程式和基于 CLI 的训练工作流的经验
- 将模型部署到 Hugging Face Hub，供他人使用
- 为更高级的微调技术打下基础知识

让我们深入探索指令微调的迷人世界！
