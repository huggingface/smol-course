# 提交你的最终项目！

是时候提交你的项目了！今年的 smol 课程将使用基于排行榜的提交系统。以下是计划：

1. 阅读本章的书面指南 ✅
2. 使用你在本章学到的知识训练一个模型。
3. 将模型推送到 Hugging Face Hub。
4. 使用 `hf jobs` 评估模型。
5. 在排行榜上开启一个拉取请求（Pull Request）。

在本页面中，我们将逐步完成每个步骤。

## 1. 阅读本章的书面指南 和 2. 使用你在本章学到的知识训练一个模型

对于第 1 章的提交，你应该阅读本章中的所有材料，并使用你学到的知识训练一个模型。大部分训练代码都在[监督微调](./3)页面上，但你需要将其与[对话模板](./2)和[使用 Hugging Face Jobs 训练](./5)页面上的代码结合起来。

## 3. 将模型推送到 Hugging Face Hub

一旦你训练了一个模型，你需要将其推送到 Hugging Face Hub 上的一个仓库。实际上，如果你在训练命令中添加 `--push_to_hub` 标志，TRL 会为你处理这个问题。所以假设你使用 `hf jobs` 训练了一个模型，那么这个参数看起来像这样：

```bash
hf jobs uv run \
    --flavor a100-large \
    --secrets HF_TOKEN \
    "trl/scripts/sft.py" \
    ...
    --push_to_hub # 这将把模型推送到 Hugging Face Hub
```

你训练的模型将在 `your-username/your-model-name` 可用。有关详细文档，请查看 `transformers` 的[检查点文档](https://huggingface.co/docs/transformers/trainer#checkpoints)。

## 4. 使用 `hf jobs` 评估模型

现在，我们需要评估模型。我们也将使用 `hf jobs` 来评估模型，并将其与 `openbench` 结合使用。我们将把评估结果推送到 Hub 上的数据集。

```sh
hf jobs uv run \ # 使用 uv 运行 hf jobs 作业
--flavor a10g-large \ # 选择机器大小
--with "lighteval[vllm]" \ # 安装带有 vllm 依赖的 lighteval
s HF_TOKEN \ # 共享 huggingface 写入令牌
lighteval vllm "model_name=<your-username>/<your-model-name>" "lighteval|gsm8k|0|0" --push-to-hub --results-org <your-username>
```

此命令将使用 `lighteval` 和 `vllm` 评估模型，并将结果保存到你定义的数据集仓库中的 Hugging Face Hub。

> [!TIP]
> 我们还没有在本课程中探索评估，但在第 2 章中，我们将更详细地探索评估。现在，我们专注于训练和提交你的模型。

## 5. 在排行榜空间上开启一个拉取请求

你现在准备好将你的模型提交到排行榜了！你需要做两件事：

1. 将你的模型结果添加到 `submissions.json`
2. 在 PR 文本中分享你的评估命令（使用 `hf jobs`）。

### 将你的模型结果添加到 `submissions.json`

在[排行榜空间](https://huggingface.co/spaces/smol-course/leaderboard/edit/main/submissions.json)上开启一个拉取请求以提交你的模型。你只需要添加模型信息和对你在上一步中创建的数据集的引用。我们将提取结果并将其显示在排行榜上。

```json
{
    "submissions": [

        ... # 现有提交

        {
            "username": "<your-username>",
            "model_name": "<your-model-name>",
            "chapter": "1",
            "submission_date": "<your-submission-date>",
            "results-dataset": "<your-results-dataset>"
        }
    ]
}
```

### 在 PR 文本中分享你的评估命令

在 PR 文本中，分享你的评估命令。例如：

```
hf jobs uv run ...
```

这将帮助我们在将其添加到排行榜之前重现你的模型评估。

### 等待 PR 合并

一旦 PR 合并，你的模型将被添加到排行榜！你可以在[这里](https://huggingface.co/spaces/smol-course/leaderboard)查看排行榜。
